The Odin Project

The Odin Project is a free online curriculum that teaches users how to become a web developer. I found it a useful tool and really enjoyed the projects it presented throughout the course work. For more information please click "here". Below are some of the projects I have completed while traversing their curriculum.

Google Remake

This is a simple remake of the google front page. The primary purpose of this exercise was to practice basic css and html.

Pixel Drawer

In this task you are asked to create a pixel drawing board in order to better understand Javascript and Jquery.

Master Mind

Master Mind is a game where the user must guess numbers until the right sequence is found. It was implemented using Ruby.

Caesar Cipher

This is a simple Caesar Cipher made using Ruby. The main focus was on Heroku deployment
